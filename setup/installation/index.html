Installation Guide¶

üì¶ Required Libraries¶

This project uses several powerful Python libraries for data analysis, machine learning, and deep learning.

Libraries Overview¶

  • Pandas

    For handling and manipulating tabular data using DataFrames Documentation

  • NumPy

    Provides fast numerical operations and multi-dimensional arrays Documentation

  • Matplotlib

    Used to create static, animated, and interactive plots Documentation

  • Scikit-learn

    A machine learning library with tools for modeling and evaluation Documentation

  • Statsmodels

    Enables statistical analysis and time series exploration Documentation

  • TensorFlow

    A powerful library for building and training deep learning models Documentation

  • Keras

    High-level API within TensorFlow for fast neural network development Documentation

  • Pydot

    Python interface for creating Graphviz DOT graphs (optional) Documentation

  • Graphviz

    Graph visualization tools used to render DOT graphs (optional) Documentation

üîß Installation Methods¶

Method 1: Using Conda (Recommended)¶

If you have Anaconda or Miniconda installed, this is the most straightforward method:

conda install -c conda-forge pandas numpy matplotlib scikit-learn statsmodels tensorflow pydot graphviz -y

Why Conda?

Conda handles complex dependencies better, especially for TensorFlow and its GPU support.

Method 2: Using pip¶

For those using standard Python installations:

# Upgrade pip first
pip install --upgrade pip

# Install core packages
pip install pandas numpy matplotlib scikit-learn statsmodels

# Install TensorFlow (CPU version)
pip install tensorflow

# Optional: For GPU support
pip install tensorflow-gpu

# Optional: For neural network visualization
pip install pydot graphviz

Method 3: Using Requirements File¶

Create a requirements.txt file:

pandas>=1.3.0
numpy>=1.21.0
matplotlib>=3.4.0
scikit-learn>=1.0.0
statsmodels>=0.13.0
tensorflow>=2.10.0
pydot>=1.4.0
graphviz>=0.20.0

Then install all at once:

pip install -r requirements.txt

üêç Setting Up Virtual Environment¶

Best Practice

Always use a virtual environment to avoid package conflicts!

Windows¶

# Create virtual environment
python -m venv hydro_env

# Activate it
hydro_env\Scripts\activate

# Install packages
pip install -r requirements.txt

# To deactivate when done
deactivate

macOS/Linux¶

# Create virtual environment
python3 -m venv hydro_env

# Activate it
source hydro_env/bin/activate

# Install packages
pip install -r requirements.txt

# To deactivate when done
deactivate

üö® Troubleshooting¶

Common Issues and Solutions¶

ImportError: No module named 'tensorflow'

Solution: Ensure you've activated your virtual environment and installed TensorFlow: 

pip install tensorflow

Graphviz not found

Solution: Graphviz requires system installation:

Windows: Download from Graphviz website

macOS: 

brew install graphviz

Linux: 

sudo apt-get install graphviz  # Ubuntu/Debian
sudo yum install graphviz       # RHEL/CentOS

Memory errors with large datasets

Solution: Consider using: - Smaller batch sizes in neural networks - Data chunking with pandas - Google Colab for free GPU access

üíª Alternative: Google Colab¶

If you prefer not to install locally, use Google Colab:

  1. Go to Google Colab
  2. Create a new notebook
  3. Most libraries are pre-installed
  4. For additional packages: 
    !pip install statsmodels pydot
